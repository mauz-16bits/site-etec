BD (Banco de Dados)

Em Banco de Dados estamos vendo sobre o Modelo Lógico de um Banco de Dados

Prof: Iury

O Modelo Conceitual de um banco de dados envolve diversos tipos de notação, como por exemplo a notação "Pé de galinha" e a notação Peter Chinn, utilizando ferramentas como o programa Dia

O Modelo Lógico consiste em relacionamento de tabelas de forma prática, utilizando ferramentas como MySQL Workbench
